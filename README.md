# 0x01.简介

这是一个由artnba3使用Java开发的超简易图书管理系统，同时也是开发者的大一Java程序设计课程的上机作业（笑）。

因为时间问题不打算使用MySQL等数据库进行改写，除账号信息外一切数据明文储存（或许以后会有时间回来改写？可能性微存x）

采用JavaFX+swing组件开发GUI，完工于2020年4月，欢迎fork并开发相关功能后pull到本仓库

以后可能会有2.0升级版本？无限咕咕咕中...
